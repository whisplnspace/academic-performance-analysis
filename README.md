# TU Dortmund Data Science Application Report

This repository contains the statistical analysis project completed for the application process of the M.Sc. Data Science program at TU Dortmund University.

## Project Overview

The project analyzes student performance data in Mathematics and Language subjects. The analysis investigates:

- Differences in scores between male and female students
- The impact of parental education level on student performance
- Statistical significance of observed differences

The project includes:
- Data preprocessing
- Exploratory data analysis (EDA)
- Data visualization
- Statistical hypothesis testing
- Interpretation of results

---

## Dataset

The dataset contains:
- Student gender
- Subject type
- Test scores
- Parental level of education

Subjects analyzed:
- Mathematics
- Language

---

## Methods Used

### Descriptive Analysis
- Summary statistics
- Grouped statistics
- Histograms
- Boxplots
- Correlation analysis

### Statistical Testing
- Shapiro-Wilk Normality Test
- Levene’s Test for Equality of Variances
- Independent Samples T-Test
- One-Way ANOVA
- Tukey HSD Post-Hoc Test
